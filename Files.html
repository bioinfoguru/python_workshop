File HandlingÂ¶

Saving and retrieving information is frequently required in developing using programs. Python has methods to read and write files. In addition, there are some library that can facilitate working with specialized file types. For reading a file, a file handler need to be initated. The open() function takes name of the file as an argument along with mode and returns the corresponding file handler. The mode here refers to the manner in which the file would be opened i.e. for reading, writing, etc. This file handler object has readlines() function that return an array of all the lines in the file. Once done with reading the contents for the file, the file handler must be closed using the close() function.

FH1 = open('test_file1.txt','r')
print(FH1)

Writing filesÂ¶

For writing content to a file the open function should be called with âwâ as the mode. When a file is opened in the write mode, a new file is created. In case there the file already exists then its contents are overwritten (without warning!).

FH_out = open('temp_file.txt', 'w')
FH_out.write("This is the first sentence.\n")
FH_out.write("This is the second sentence.\n")
print("Writing to file", file=FH_out)
FH_out.close()

FH2 = open('temp_file.txt','r')
for lines in FH2:
    print(lines)
FH2.close()

FH_out.write("This is the \t fourth sentence.\n")

FH3 = open('temp_file.txt','r')
for lines in FH3:
